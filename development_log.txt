## 技術仕様

### ゲーム画面構成
```
┌─────────────────┬─────────┐
│   ゲームエリア    │ UIエリア │
│   853x720      │ 200x720 │
│   (宇宙背景)     │ (青背景) │
└─────────────────┴─────────┘
```

### 爆弾システム
- 1ライフあたり: 2個まで
- 被弾時: 2個にリセット
- 戦略的使用が重要

### 宇宙背景システム
- Star: トゥインクル効果
- Nebula: パルス効果
- Planet: 回転効果
- ShootingStar: 軌跡効果

## 開発環境

### 必要な依存関係
```
pygame==2.6.1
numpy (音声生成用)
```

### ファイル構成
```
QGamen_DanmakuShooting/
├── main.py                 # エントリーポイント
├── src/                    # ソースコード
│   ├── game.py            # メインゲームクラス
│   ├── player.py          # プレイヤークラス
│   ├── enemy.py           # 敵クラス
│   ├── bullet.py          # 弾丸クラス
│   ├── ui.py              # UIクラス
│   ├── effects.py         # エフェクトクラス
│   ├── items.py           # アイテムクラス
│   ├── ranking.py         # ランキングクラス
│   ├── audio_manager.py   # 音声管理クラス
│   ├── audio_generator.py # 音声生成クラス
│   ├── font_manager.py    # フォント管理クラス
│   └── space_background.py # 宇宙背景クラス
├── assets/                 # アセット
│   └── audio/             # 音声ファイル
├── rankings.json          # ランキングデータ
└── requirements.txt       # 依存関係
```
